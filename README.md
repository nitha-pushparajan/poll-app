## Overview

The poll app is created with react, typescript, redux, tailwind css, and storybook.

## Setup

Install the dependencies and devDependencies and start the server.

```
  npm install
  npm run start
```

## Scripts

### start the development server

```
  npm run start
```

### format and fix

```
  npm run format
```

### Run storybook

```
  npm run storybook
```

### Run tests

```
  npm run test
```
